# 图中两个点之间的路线
## 题目描述

176
给出一张有向图，设计一个算法判断两个点 s 与 t 之间是否存在路线。
```
	A----->B----->C
	 \     |
	  \    |
	   \   |
	    \  v
	     ->D----->E
			 
样例 1:
输入:s = B and t = E,
输出:true

样例 2:
输入:s = D and t = C,
输出:false
```

## 题目分析
搜索算法的最直观应用
可以用BFS，DFS
分别实现一遍

## 题目解答
```python
# DFS
"""
Definition for a Directed graph node
class DirectedGraphNode:
    def __init__(self, x):
        self.label = x
        self.neighbors = []
"""


class Solution:
    """
    @param: graph: A list of Directed graph node
    @param: s: the starting Directed graph node
    @param: t: the terminal Directed graph node
    @return: a boolean value
    """

    def hasRoute(self, graph, s, t):
        # write your code here
        return self.dfs(graph, s, t, set())
		return self.bfs(graph, s, t)
        
    def dfs(self, graph, s, t, visited):
        if s == t:
			# base case
            return True
        for node in s.neighbors:
            if node in visited:
                continue
            visited.add(node)
            if self.dfs(graph, node, t, visited):
                return True
        return False
    
	def bfs(self, graph, s, t):
        queue = []
        queue.append(s)
        visited = set()
        
        while queue:
            n = len(queue)
            for i in range(n):
                node = queue.pop(0)
                if node in visited:
                    continue
                if node == t:
                    return True
                visited.add(node)
                queue.extend(node.neighbors)
        return False
```
